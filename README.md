# T-Bank Queue — Full Project

This is a demo queue system (Node.js + Express + SQLite) with a T-Bank themed frontend.
It includes:
- server.js (Express API + SSE + SQLite using better-sqlite3)
- public/ (frontend UI and admin pages)
- data.db created automatically on first run
- package.json (dependencies and start script)

## Quick start

1. Unpack the archive.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start server:
   ```bash
   npm start
   ```
4. Open in browser:
   - User UI: http://localhost:3000/
   - Admin: http://localhost:3000/admin.html

Settings: change global settings in Admin → Global settings.
